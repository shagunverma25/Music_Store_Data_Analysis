# SQL_Project_Music_Store_Data_Analysis
SQL project to analyze online music store data

This project analyzes a digital music store database using SQL to extract business insights and help the store understand its business growth by answering simple questions.
The goal of this project is to simulate how a real company can use SQL to make data-driven decisions.

##  Database and Tools
- PostgreSQL  
- pgAdmin 4  
- SQL
